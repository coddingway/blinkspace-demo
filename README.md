# BlinkSpace — Concept Redesign

A high-fidelity, single-page concept redesign for **BlinkSpace** — a strategic advisory firm operating at the intersection of space technology, defence, policy, and capital.

## Purpose

Reimagining BlinkSpace's digital presence with a premium editorial aesthetic that matches the gravity of the sector they serve — satellite tech, national security, and deep-tech investment.

## Live Demo

[demo.amritpodder.dev](https://demo.amritpodder.dev)

## What's Built

- Full single-page website: Hero, About, Services, Approach, Team, Contact
- Dark editorial design language (DM Serif Display + DM Sans)
- Autoplay background video in the Approach section
- Smooth scroll, fade-in animations, responsive layout
- Zero dependencies — pure HTML, CSS, and vanilla JS

## Tech Stack

| Layer | Technology |
|-------|------------|
| Markup | HTML5 (single file) |
| Styling | Vanilla CSS (custom properties, grid, clamp) |
| Interactivity | Vanilla JS (IntersectionObserver, scroll effects) |
| Typography | DM Serif Display · DM Sans · DM Mono (Google Fonts) |
| Hosting | Vercel |

## Deployment

Deployed as a static site on Vercel. SPA routing handled via `vercel.json`.
